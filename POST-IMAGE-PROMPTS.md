## How to use these

Paste any prompt into **Midjourney**, **DALL-E 3**, or **Adobe Firefly**.

**Midjourney:** append `--style raw --ar 1:1 --stylize 750 --no hands fingers`

**If the result is too photorealistic:** add *"flat graphic, no photorealism, no 3D rendering, no photography"*

**If colours bleed:** add *"strict colour separation, screen-print halftone, two-colour only"*

**If text appears:** add *"absolutely no typography, no letters, no words, no numbers"*

**If compositions look similar:** explicitly name where the photographic element sits — "lower-left quadrant", "bleeds off the right edge", "upper-right corner". Vary this per generation.

Once generated, save as `/public/images/[post-slug].jpg` and add to the post's frontmatter:
```yaml
ogImage: "/images/your-post-slug.jpg"
```
